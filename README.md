# Chairfighter

A 2D action platformer where a basic chair defeats boss chairs and unlocks transformations.

## MVP Scope

Basic Chair → Recliner Baron → Armchair unlock → Armchair grapple

## Setup

1. Install Godot 4.x: https://godotengine.org/download/
2. Open the project folder in Godot
3. Press F5 to run the prototype

## Known Limitations

- Godot CLI headless validation available
- Placeholder art only
- Controller-first input with keyboard support

## Structure

- `scenes/` — Godot scenes
- `scripts/` — GDScript files
- `assets/` — Placeholder art and audio
